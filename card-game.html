Card Game Program

This Was my first attempt at any sort of Game development. The Card Game I started with was GO Fish. In this program I use, a class with functions to control player turns and computer turns.

The Go Fish Game includes a variety of functions. I use the itertools library to create a deck of cards and I use to the counter() function to move through the deck while counting what number card I am on.
