My Very First Web App

I titled this project "Data Bank". The idea for Data Bank came about when I couldn't access the information in my bank account remotely and had no way of viewing how much money that I had. So I set out to create an application that solved a complex problem with ease.

The app only keeps up with numbers that you enter. You tell it that you have x amount and saves that number x locally to be pulled when access the page again.

The application uses three functions in javascrip. Addition, Subtraction, and Window Local Storage.
